# arithmeticByPhp 算法与设计模式整理

```
├── sort // 这排序算法
│   ├── sortQuick.php  // 快速排序
│   ├── sortSimpleBucket.php // 简单桶排序
│   └── sortBubbling.php // 冒泡排序
├── search // 搜索算法
│   ├── depthFirstSearch.php  // 深度优先搜索（Depth First Search）
├── designMode // 设计模式
│   ├── single.php  // 单列模式
│   ├── ioc_di.php  // 控制反转（IoC）和依赖注入（DI）
│   ├── factory.php  // 简单工厂模式
```
